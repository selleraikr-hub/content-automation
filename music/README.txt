[ 배경음(BGM) 넣는 법 ]

짤영상 배경음은 이 폴더(music/)의 음악 파일 중 게시할 때마다 하나를 랜덤으로 골라 씁니다.

■ 곡 추가
  무료·저작권 프리 mp3를 이 폴더에 넣기만 하면 자동으로 사용됩니다.
  파일명은 자유 (단 bgm_ 로 시작하면 제외 — 옛 합성음 예약어).
   - YouTube 오디오 보관함: studio.youtube.com → 오디오 보관함
   - Pixabay Music: pixabay.com/music (로그인 없이 다운로드)

■ 곡이 없으면 무음으로 생성됩니다.

■ 반드시 "상업적 사용 가능 / 저작권 프리" 곡만 사용하세요.
  (유행 가요·팝 금지 — 스레드에서 음소거/삭제 위험)
