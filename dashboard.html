📝 네이버 블로그 자동 발행

주제 입력 → ① 수익형 키워드 추출 → ② 클릭 제목 → ③ SEO 본문(네이버 좋은콘텐츠 5원칙) → ④ 이미지(Unsplash) → ⑤ 자동 발행.

사용법

1
위 실행 탭에서 주제 입력 후 블로그 자동 발행 클릭. (터미널로는 node run_all.js "주제")
2
크롬이 열리고 제목·본문·이미지가 자동 입력된 뒤 발행됩니다.
3
처음엔 임시저장 체크 후 실행해 결과를 확인하는 걸 권장. (--draft)

결과물

  • post.json 생성된 글
  • keywords_*.md 키워드 분석
  • naver_blog_log.txt 발행 로그

이미지 품질

.env의 UNSPLASH_ACCESS_KEY가 있으면 주제 맞춤 고화질 사진, 없으면 대체 이미지가 들어갑니다.

🎵 틱톡 카드뉴스 영상 자동 업로드

블로그 글(post.json)을 → 카드뉴스 PNG → 세로 9:16 영상(mp4) → TikTok Studio 웹 업로드 + 캡션 자동 입력.

사전 준비

  • ffmpeg 설치: winget install Gyan.FFmpeg (설치 후 새 CMD)
  • 틱톡 로그인: 실행 탭의 "틱톡 로그인" 1회

사용법

1
실행 탭에서 주제 입력 → 틱톡 발행 클릭. (node tiktok_all.js "주제")
2
기본은 검토 모드 — 영상·캡션이 올라간 채 게시 직전에 멈춥니다. 확인 후 빨간 게시 직접 클릭.
3
완전 자동으로 게시까지 하려면 틱톡 자동 게시 체크 후 실행. (--publish)
🎵 BGM: music/ 폴더에 저작권 프리 mp3를 넣으면 영상에 자동으로 배경음악(끝 페이드아웃)이 들어가요. 없으면 무음. 저작권 있는 곡은 틱톡에서 음소거될 수 있어요.

⚙️ 설치 · 설정

1) 필수 준비

  • Node 18+ (node -v), Playwright (npm install + npx playwright install chromium)
  • ffmpeg(틱톡 영상용): winget install Gyan.FFmpeg

2) API 키 (.env)

  • ANTHROPIC_API_KEY — 글 생성(필수). console.anthropic.com
  • UNSPLASH_ACCESS_KEY — 이미지(선택). unsplash.com/developers

3) 로그인 세션 (최초 1회)

  • 네이버: 실행 탭 "네이버 로그인"
  • 틱톡: 실행 탭 "틱톡 로그인"
.env와 세션 파일에는 민감정보가 있어요. 절대 공유·업로드하지 마세요.

🛠 문제해결

글 생성이 "JSON을 찾지 못함"

응답이 길어 잘린 경우. config.js의 MAX_TOKENS를 늘리세요(현재 8000).

모델 오류(404 model)

.env의 CLAUDE_MODEL을 콘솔에서 쓸 수 있는 모델명으로 변경.

발행/업로드 버튼이 안 눌림

네이버·틱톡 화면 구조가 바뀐 경우. naver_blog_log.txt / tiktok_log.txt를 확인해 어디서 막혔는지 보고 셀렉터를 조정하세요.

ffmpeg 없음

winget install Gyan.FFmpeg 후 새 CMD 창에서 ffmpeg -version 확인.
