# 쿠팡 파트너스 API 발급 가이드

## 승인 조건 (핵심)
1. **누적 수익 10만원 이상** (제휴 수익 발생 기록) ← 가장 중요
2. 활동 중인 채널(블로그/SNS 등) 등록 + 트래픽·콘텐츠 품질
3. API 활용 계획 작성
→ 방금 가입한 새 계정(수익 0)은 발급 불가. "생성" 버튼 비활성.
→ 승인 소요: 보통 1~3 영업일 (조건 충족 후 신청 시)

## 지금 (API 없이) 운영법
1. 쿠팡 파트너스 → **링크 생성**에서 상품 제휴링크 만들기 (승인 전에도 가능)
2. 그 링크를 --url 로 발행:
   node threads_copy.js "크록스" --url "만든_제휴링크"     (복붙용)
   node threads_post.js "크록스" --url "만든_제휴링크"      (스레드 자동, 검토모드)
3. 스레드/블로그에 꾸준히 올려 누적 수익 10만원 쌓기

## 나중 (수익 10만원+ 되면) 자동화
1. 파트너스 → 추가 기능 → 파트너스 API → "생성" (활성화됨)
2. Access Key / Secret Key 복사
3. .env 에 입력:
   COUPANG_ACCESS_KEY=발급키
   COUPANG_SECRET_KEY=발급키
4. 이후엔 링크 없이 키워드만:  node threads_copy.js "크록스"  → 자동 매칭

## 승인 확률 높이기
- "내 정보 관리"에 채널(블로그/스레드) 등록 + 파트너스 링크 게시 화면 스크린샷 업로드
- 꾸준한 게시 + 실제 클릭/구매 발생
